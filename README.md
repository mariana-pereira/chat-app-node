## Building

You'll need [Node.js](https://nodejs.org) installed on your computer in order to build this app.

```bash
git clone https://github.com/mariana-pereira/chat-me-node
$ cd chat-app
$ yarn install
$ yarn dev
```

Runs the app in the development mode.
